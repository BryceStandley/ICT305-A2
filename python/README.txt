ICT305 Assignment 1 README - PinkFluffyUnicorns

Additional python module Import_ipynb is required to launch the application

To see all other required python modules, see requirements.txt

When first opening the application on Windows:
    Right click and launch  setup_app.bat   as an administrator to install any python modules within the requirements.txt file
    Note. The script assumes python is installed and the command 'py' is available 

To launch the web application on Windows:
    Launch  start_app.bat
    The batch script launches the master ipynb file with bokeh serve and shows the application in the default browser
